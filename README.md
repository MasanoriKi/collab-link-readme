### 注文書・注文請書管理

**一覧画面**
- 月別に切り替えて表示
- 発行済みの注文書がカードで表示される
- ステータスは「確認待ち」「送付済み」「承認済み」で表示される

**詳細画面**
- 注文書の内容を確認できる
- 印刷ボタンから帳票を印刷できる
- 承認済みの場合は「すべて印刷」と「請書のみ印刷」が使える

**承認**
- 未承認の注文書は「詳細・承認」から開いて承認できる
- 承認すると注文請書が発行され、ステータスが承認済みになる

---

### 日程調整（責任者のみ）

**一覧画面**
- 月別に切り替えて表示
- 未回答が上、回答済みが下にまとまって表示される
- 未回答のある月には黄色のドット（●）が付く

**単体回答**
- カレンダーで作業可能日を選択して送信
- 選択した日にチップ（タグ）が表示される
- 既に他の物件で確定済みの日を選ぶと警告が出る（送信は可能）

**一括回答**
- 複数物件をまとめて一画面で回答できる
- 各物件で日付を選択し、まとめて送信
- 日付未選択の物件はスキップされる
- 他の物件と同じ日を選択すると警告が出る（送信は可能）

**日程のキャンセル**
- 回答済みの物件の詳細ページから、確定した作業日を日付単位でキャンセルできる
- キャンセル理由の入力が必須
- **代替日の再調整を希望する場合**（代替日希望にチェックを入れる）
  - 社員へ「要対応」のキャンセル通知メールが届く（日程調整ページへのリンク付き）
  - 協力会社側の一覧に「送信待ち」として表示され、社員が新しい調整を送るまで待機状態になる
  - 社員が再送すると通常の未回答として表示される
- **代替日を希望しない場合**
  - 社員へキャンセル通知メールのみ届く
  - 一覧から該当物件が消える
    - 有効日が残っている場合はキャンセル日のみキャンセルされる

### 日程調整送信

**一覧画面**
- 年月をピッカーで選択（デフォルトは2ヶ月後）
- 作業タイプでフィルタリング可能（フィルター中は絞り込んだ分だけ送信）
- 会社名で検索フィルター
- 未登録の協力会社・メール未設定の場合は送信ボタンが無効

**送信単位**
- 会社単位でまとめて送信（表示中のアイテムのみ）
- 作業予定ごとに個別送信
- 画面上部「全社に送信」で表示中の未送信会社を一括送信

**送信すると**
- 協力会社の代表者メールに日程調整依頼メールが届く
- 協力会社側の日程調整画面に物件が追加される

**協力会社からキャンセルされた場合**
- キャンセル通知メールが届く
- 代替日希望ありの場合は件名に「要対応」が付き、メール内に日程調整ページへのリンクが表示される
- 一覧画面で該当会社のカードに「再調整依頼あり（未送信）」バッジが表示される
- 新しい日程調整を送信すると協力会社側に再度通知される

### 注文書発行

**一覧画面**
- 月別に切り替えて表示
- 確定済み作業がある協力会社だけ表示される
- 会社名で絞り込みできる
- 会社ごとに「確定件数」「注文書の状態」「操作」が表示される

**状態表示**
- 未発行の会社は「未発行」と表示される
- 発行済みの注文書が1件のときは注文番号と状態が表示される
- 複数回発行している場合は「◯件発行済み」と表示される

**確認・発行**
- 「確認・発行」から注文書プレビューを開ける
- 未発行の場合は「発行して印刷」ボタンが表示される
- 発行済みで未承認の場合は「印刷のみ」ボタンが表示される
- 承認済みの場合は「すべて印刷」と「請書のみ印刷」が表示される

**確認**
- 発行済み注文書は「確認」から内容確認できる
- 複数発行している場合は対象の注文番号を選んで開ける
- 承認済みの注文書では「承認取り消し」ができる（未実装）
